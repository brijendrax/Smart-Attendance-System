# Smart Attendance System

Smart Attendance System is a face recognition-based attendance management project built using Python, OpenCV, Flask, HTML, and CSS. The system captures student faces through a webcam, recognizes them using the LBPH algorithm, and automatically marks attendance with date and time in a CSV file.

## Features

* Face Recognition Attendance
* Automatic Attendance Marking
* Flask Web Dashboard
* Webcam Integration
* Date & Time Tracking
* Duplicate Attendance Prevention
* Modern User Interface

## Technologies Used

* Python
* OpenCV
* Flask
* HTML
* CSS

## How to Run

1. Install required libraries
2. Run `python app.py`
3. Open `http://127.0.0.1:5000`
4. Capture faces, train model, and start attendance
